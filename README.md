# nepu-大学物理实验计算器
**此实验计算器仅适用于东北石油大学大学物理实验课程**  
由于GitHub中仅支持英文文件名所以以后的计算器文件均用实验XX中的XX代替  
例如：**实验B01 金属丝弹性模量的测量**在计算器下载界面中中的命名为**B01**  
所有计算器的使用方法均为在**绿色区域**填写数据，在**红色区域**横纵标识获取对应结果
